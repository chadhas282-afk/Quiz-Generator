# Awesome Quiz Application

A dynamic, responsive, and interactive Quiz Application built using **HTML5, CSS3, and Vanilla JavaScript**. This project features a sleek UI with countdown timers, progress bars, and real-time score tracking.



üöÄ Features

* **Interactive UI:** Smooth transitions and animations using CSS3.
* **Timed Questions:** Each question has a 10-second limit.
* **Visual Feedback:**
* **Tick/Cross Icons:** Instant visual cues for correct and incorrect answers.
* **Auto-Reveal:** If the timer runs out or the user picks the wrong answer, the correct one is automatically highlighted.


* **Progress Tracking:** A dynamic "Time Line" at the top and a question counter at the bottom.
* **Dynamic Scoring:** A result screen that provides custom messages based on user performance.
* **Fully Responsive:** Designed with a mobile-first approach using the Poppins font.


üìÇ Project Structure

* `index.html`: The structural backbone of the application.
* `style.css`: Contains all the styling, including the layout for the info box, quiz container, and result box.
* `questions.js`: A dedicated data file containing the array of questions, options, and correct answers.
* `script.js`: The core logic handling the timer, scoring, DOM manipulation, and state changes.

üìù Rules of the Quiz

1. You have only **10 seconds** per question (configured to 10s in the script).
2. Once you select an answer, it cannot be undone.
3. Options are disabled once the timer reaches zero.
4. You cannot exit the quiz while it is in progress.
5. Scores are calculated based on the number of correct answers.
