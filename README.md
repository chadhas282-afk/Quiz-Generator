
# Awesome Quiz Application

A sleek, dynamic, and interactive Quiz Application built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This application features a real-time countdown timer, a dynamic progress bar, and instant feedback on user answers.



---

## üöÄ Features

* **Interactive UI:** Smooth transitions and a clean interface using the Poppins font.
* **Timed Challenges:** Users have 10 seconds to answer each question before the time runs out.
* **Dynamic Scoring:** Track your score in real-time; the final result screen provides feedback based on your performance.
* **Visual Feedback:** * **Correct/Incorrect Highlighting:** Green and red styling for instant feedback.
    * **Auto-Selection:** If time runs out or an incorrect answer is chosen, the correct answer is automatically revealed.
    * **Tick & Cross Icons:** Integrated FontAwesome icons for a polished look.
* **Responsive Progress:** A linear "Time Line" at the top and a question counter at the bottom.

* `style.css`: All styling, including the glassmorphism-inspired effects and layout.
* `questions.js`: A dedicated data file containing the questions, options, and answers.
* `script.js`: The engine that drives the quiz logic, timer, and score calculation.

---

## üìù Rules of the Quiz

1.  You have only **10 seconds** per each question.
2.  Once you select your answer, it can't be undone.
3.  You can't select any option once time goes off.
4.  You can't exit from the Quiz while you're playing.
5.  You'll get points on the basis of your correct answers.
